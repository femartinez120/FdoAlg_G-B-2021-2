Estructurar la web con HTML

Para crear sitios web, debe conocer HTML , la tecnología fundamental que se utiliza para definir la estructura de una página web. HTML se utiliza para especificar si su contenido web debe reconocerse como un párrafo, lista, encabezado, enlace, imagen, reproductor multimedia, formulario o uno de los muchos otros elementos disponibles o incluso un nuevo elemento que defina.

¿Quiere convertirse en un desarrollador web front-end? Hemos elaborado un curso que incluye toda la información esencial que necesita para trabajar hacia su objetivo.

Prerrequisitos

Antes de comenzar con este tema, debe tener al menos una familiaridad básica con el uso de computadoras y el uso pasivo de la web (es decir, simplemente mirarla, consumir el contenido). Debe tener un entorno de trabajo básico configurado como se detalla en Instalación de software básico y comprender cómo crear y administrar archivos, como se detalla en Manejo de archivos ; ambos son partes de nuestro Módulo para principiantes completo de Introducción a la web .

Se recomienda que trabaje en Introducción a la web antes de intentar este tema. Sin embargo, esto no es absolutamente necesario; gran parte de lo que se cubre en el artículo de conceptos básicos de HTML también se cubre en nuestro módulo Introducción al HTML , aunque con mucho más detalle.

Después de aprender HTML, puede pasar a aprender sobre temas más avanzados, como:

CSS y cómo usarlo para diseñar HTML (por ejemplo, altere el tamaño del texto y las fuentes utilizadas, agregue bordes y sombras, diseñe su página con múltiples columnas, agregue animaciones y otros efectos visuales).
JavaScript , y cómo usarlo para agregar funcionalidad dinámica a las páginas web (por ejemplo, encontrar su ubicación y trazarla en un mapa, hacer que los elementos de la interfaz de usuario aparezcan / desaparezcan al alternar un botón, guardar los datos de los usuarios localmente en sus computadoras, y mucho más.)
